# Newsweek

![screenshot](img/newsweek.png)

> This is a clone of newsweek webpage using HTML CSS, and bootstrap.  

> The goal here is to make a responsive website using bootstrap fundamentals.

## Built With

- HTML5, CSS3
- Bootstrap
- Windows/Linux
